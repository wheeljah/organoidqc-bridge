📖 OrganoidQC Web — 연구자 매뉴얼

🔬 서비스 소개

OrganoidQC Web은 오가노이드 배양 품질을 빠르게 평가하고 레퍼런스 데이터와 비교할 수 있는 연구자 전용 웹 도구입니다. OrganoID AI 분석 엔진과 연동하여 현미경 이미지에서 직접 QC 지표를 추출합니다.

⚡ 주요 기능 및 장점

  • 즉시 QC 분석 — 이미지 업로드 후 5초 내 Size CV, Circularity, Viability 지표 자동 산출
  • OSI 가이드라인 벤치마킹 — 국제 오가노이드 표준(OSI) 기준과 자동 비교 및 Pass/Fail 판정
  • OrganoID AI 연동 — Bridge 서버를 통해 딥러닝 기반 세그멘테이션으로 개별 오가노이드 수 정밀 검출
  • Real Data 비교 — 자체 CSV 데이터 또는 Supabase DB와 연결하여 시뮬레이션 vs 실측 비교 차트 생성
  • 다중 장기 지원 — 장(Intestine), 간(Liver), 뇌(Cortical) 오가노이드별 최적화된 QC 기준 적용
  • 웹 기반, 설치 불필요 — 브라우저만 있으면 어디서든 접근 가능

🚀 사용 방법 (Step-by-step)

Step 1. 왼쪽 사이드바에서 장기 종류와 배양 일수를 선택합니다.
Step 2. 로컬 이미지 업로드 영역에 현미경 사진을 드래그하거나 "파일 선택"을 클릭합니다. (또는 웹 URL 이미지 사용)
Step 3. "🔬 이미지 분석 시작" 버튼을 클릭합니다. 픽셀 알고리즘이 자동 실행되며, Bridge 서버가 연결된 경우 OrganoID AI 분석도 자동으로 실행되어 알고리즘 비교 섹션이 표시됩니다.
Step 4. QC 지표 카드(Size CV, Circularity, Viability, 오가노이드 수)와 OSI 벤치마킹 테이블을 확인합니다. Bridge 연결 시 아래 알고리즘 비교 섹션에서 픽셀 알고리즘 vs OrganoID 결과를 나란히 비교할 수 있습니다.
Step 5 (선택). 📂 CSV 파일을 업로드하거나 수동으로 Real Data를 입력하고 "비교 차트 업데이트"를 눌러 실측 데이터와 비교합니다.
Step 6 (선택). Supabase를 연결하면 여러 연구자의 데이터를 공유 DB에 저장하고 불러올 수 있습니다.
